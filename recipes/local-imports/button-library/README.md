# Button Library Demo

This folder demonstrates a **self-contained** Reactive MD project with local imports.

## Structure

```
button-library/
├── demo.md       # This file - the interactive document
├── Button.jsx    # Reusable button component
└── styles.css    # Custom CSS styles
```

## Usage

Open `demo.md` in VS Code and press `Cmd+K P` for Interactive Preview.

## The Components

See the `demo.md` file for the live examples.
