## Installation

1. Open VS Code
2. Press `Cmd+Shift+X` (Mac) or `Ctrl+Shift+X` (Windows/Linux)
3. Search for "JSX Preview"
4. Click Install

Or install from the command line:
```bash
code --install-extension anthropic.jsx-preview
```
